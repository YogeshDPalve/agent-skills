# GitHub Commit Writer Skill

This skill automates writing and pushing commit messages to GitHub repositories. It ensures consistent commit formatting and provides suggestions based on the changes made in the repository.

## Features
- Analyze changes in the repository to generate relevant commit messages.
- Follow a customizable commit message style guide.
- Automate the staging and committing process.

## Usage
- Place this skill in your skills directory.
- Configure your style preferences in the skill settings.
- Call the skill to generate and push commit messages.

## Installation
1. Add this skill to your skill manager.
2. Configure your GitHub credentials for access.
3. Start automating your commits!
